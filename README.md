# Student Assignment Tracker (ServiceNow)

## Overview
The Student Assignment Tracker is a custom scoped application engineered on the ServiceNow platform. It provides a centralized, role-based system for educational institutions to manage courses, track student submissions, and streamline the grading process.

## Key Features
* **Role-Based Workspaces:** Custom Service Portals tailored specifically for Students, Teachers, and Heads of Departments (HOD).
* **Interactive Dashboards:** Utilizes custom widgets, relational page architecture, and premium CSS/UI styling.
* **Data Management:** Custom tables and cross-scope functionality to securely manage user profiles, assignments, and grades.

## Technical Details
* **Platform:** ServiceNow
* **Architecture:** Scoped Application
* **UI/UX:** Service Portal, Custom Widgets, HTML/CSS
* **Version Control:** Integrated with GitHub via ServiceNow Source Control
